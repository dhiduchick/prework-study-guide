HTML

  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating webpages.
  • The head element conatina information about the webpage.
  • The body ekement represents the visible contenct shown to the user.
  • Inside the head we can use different metadata tags such as title,style,link,meta, and script.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • '*' applys a rule to all elements visible on the page.
  • CSS stands for Cascasding Style Sheets
  • CSS is the language used to describe the presentation of a website that is created with HTML.
  • With CSS you can add style suchs as texts,fonts,and spacing.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and merge any commits from the tracking remote branch
  • Git is a free open source version control system that allows web developers to collaborate on projects without overwriting eachothers work.
  • Git allows developers to revert to the previous versions of the code if a problem is discovered.
  • Github is the largest provider of internet hosting for software and webdevelopment code, and provides the version control functionally of Git in a central repository.

JavaScript

  • JavaScript is a lightweight programming language that is used to provide logic and interactivity for websites.
  • JavaScript uses the Script element to apply internal stylesheets to the HTML.
  • A variable is a named container that allows is to store date in our code.
  • Control flow is the order in which a computer executes code in a script.
